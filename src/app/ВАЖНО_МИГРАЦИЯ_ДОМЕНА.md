## ⚠️ ОБЯЗАТЕЛЬНО ВЫПОЛНИТЕ ПОСЛЕ ПЕРЕНОСА DNS

### 1. Настройки Supabase (5 минут)

**Путь:** Supabase Dashboard → Authentication → URL Configuration

**Измените:**
```
Site URL: https://coffeenechai.ru
```

**Добавьте в Redirect URLs:**
```
https://coffeenechai.ru/**
```

❗ **БЕЗ ЭТОГО НЕ БУДЕТ РАБОТАТЬ РЕГИСТРАЦИЯ И ВХОД**

---

### 2. Тестирование (10 минут)

После настройки DNS и Supabase проверьте:

**РОЗНИЦА:**
- Регистрация нового пользователя
- Оформление и оплата заказа
- Переход на страницу успеха после оплаты

**ОПТ:**
- Вход по логину/паролю
- Создание заказа
- Telegram уведомление
