# ⚡ Быстрое исправление: "Failed to fetch"

## 🔴 Ошибка
```
Не удалось подключиться к серверу: Failed to fetch
```

## ✅ Решение (3 шага)

### 1️⃣ Проверьте сервер
Откройте в браузере:
```
https://pkhinqiplfezrzvsqgwo.supabase.co/functions/v1/make-server-aa167a09/health
```

**Должны увидеть:**
```json
{"status":"ok","timestamp":"...","service":"nechai-server"}
```

**Если ошибка 404** → Сервер не развернут, переходите к шагу 2

### 2️⃣ Разверните Edge Function

1. Откройте [Supabase Dashboard](https://supabase.com/dashboard)
2. Выберите проект: `pkhinqiplfezrzvsqgwo`
3. Меню: **Edge Functions** → **server**
4. Кнопка: **Deploy** (или **Redeploy**)
5. ⏰ Подождите 30-60 секунд
6. 🔄 Обновите страницу админки

### 3️⃣ Запустите диагностику

1. На странице админки при ошибке появится блок "Диагностика сервера"
2. Нажмите кнопку **"Запустить диагностику"**
3. Дождитесь результатов тестов
4. Если все ✅ зеленые - нажмите "Попробовать снова"
