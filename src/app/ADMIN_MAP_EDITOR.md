# 🗺️ Интерактивная карта для точного позиционирования точек продаж

**Дата:** 27 января 2026  
**Статус:** ✅ РЕАЛИЗОВАНО

---

## 🎯 Проблема

При добавлении точек продаж через автокомплит адресов, координаты могут быть **неточными**:
- 📍 Метка может стоять на соседнем здании
- 📍 Координаты могут быть на несколько метров в стороне
- 📍 Кофейня на первом этаже, а метка на крыше

---

## ✨ Решение

Добавлена **интерактивная карта с перетаскиваемым пином** прямо в админ-панель!

### Возможности:

1. **Перетаскивание метки** 🖱️
   - Хватаете метку и тащите в нужное место
   - Координаты обновляются автоматически
   - Адрес обновляется через обратное геокодирование

2. **Клик по карте** 👆
   - Кликните в любом месте карты
   - Метка переместится туда
   - Координаты и адрес обновятся

3. **Автоматическое центрирование** 🎯
   - Карта автоматически центрируется на выбранной точке
   - При выборе из автокомплита - карта показывает эту точку

4. **Обратное геокодирование** 🔄
   - При перемещении метки адрес обновляется автоматически
   - Вы всегда видите актуальный адрес

## 🛠️ Технические детали

### Используемые API:

#### 1. Яндекс.Карты API 2.1
```javascript
// Загрузка API
<script src="https://api-maps.yandex.ru/2.1/?apikey=XXX&lang=ru_RU"></script>

// Инициализация карты
const map = new window.ymaps.Map(container, {
  center: [59.9386, 30.3141], // [lat, lon]
  zoom: 15,
  controls: ['zoomControl', 'searchControl']
});
```

#### 2. Перетаскиваемая метка
```javascript
const placemark = new window.ymaps.Placemark([lat, lon], {
  balloonContent: 'Перетащите метку в нужное место'
}, {
  preset: 'islands#redDotIcon',
  draggable: true  // ← Ключевой параметр!
});
```

#### 3. Обработка перетаскивания
```javascript
placemark.events.add('dragend', function (e) {
  const coords = placemark.geometry.getCoordinates();
  
  // Обновляем координаты
  setSelectedCoords({
    lat: coords[0],
    lon: coords[1]
  });
  
  // Обратное геокодирование
  window.ymaps.geocode(coords).then((res) => {
    const firstGeoObject = res.geoObjects.get(0);
    const newAddress = firstGeoObject.getAddressLine();
    setAddress(newAddress);
  });
  
  toast.success('📍 Координаты обновлены');
});
```

#### 4. Обработка клика по карте
```javascript
map.events.add('click', function (e) {
  const coords = e.get('coords');
  
  // Перемещаем метку
  placemark.geometry.setCoordinates(coords);
  
  // Обновляем координаты и адрес
  setSelectedCoords({ lat: coords[0], lon: coords[1] });
  
  // Обратное геокодирование
  window.ymaps.geocode(coords).then((res) => {
    const newAddress = res.geoObjects.get(0).getAddressLine();
    setAddress(newAddress);
  });
});
```

## 🔄 Lifecycle карты

### Создание карты:
```
useEffect(() => {
  if (!showMap || !ymapsLoaded || mapInstance) return;
  
  // Создаем карту
  const map = new window.ymaps.Map(...);
  const placemark = new window.ymaps.Placemark(...);
  
  // Добавляем метку
  map.geoObjects.add(placemark);
  
  // Сохраняем инстансы
  setMapInstance(map);
  setPlacemarkInstance(placemark);
  
  // Cleanup
  return () => {
    if (map) {
      map.destroy();
    }
  };
}, [showMap, ymapsLoaded]);
```

### Обновление метки:
```
useEffect(() => {
  if (placemarkInstance && selectedCoords) {
    // Обновляем положение метки
    placemarkInstance.geometry.setCoordinates([
      selectedCoords.lat,
      selectedCoords.lon
    ]);
    
    // Центрируем карту
    mapInstance.setCenter([...], 15);
  }
}, [selectedCoords, placemarkInstance]);
```
