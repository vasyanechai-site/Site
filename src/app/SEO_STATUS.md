1. Зарегистрируйтесь: https://search.google.com/search-console
2. Добавьте сайт coffeenechai.ru
3. Получите verification код
4. Добавьте в `/index.html`:
   ```html
   <meta name="google-site-verification" content="ВАШ_КОД" />
   ```
5. Отправьте sitemap: https://coffeenechai.ru/sitemap.xml
6. Запросите индексацию главной страницы

#### Яндекс.Вебмастер
1. Зарегистрируйтесь: https://webmaster.yandex.ru
2. Добавьте сайт coffeenechai.ru
3. Получите verification код
4. Добавьте в `/index.html`:
   ```html
   <meta name="yandex-verification" content="ВАШ_КОД" />
   ```
5. Отправьте sitemap: https://coffeenechai.ru/sitemap.xml

## 🔧 Поддержка

Для мониторинга индексации:
1. Google Search Console → Performance → Search Results
2. Яндекс.Вебмастер → Поисковые запросы
3. Проверка вручную: site:coffeenechai.ru в Google/Яндекс

Если через 2 недели сайт не индексируется:
1. Проверьте отсутствие noindex
2. Убедитесь что robots.txt правильный
3. Запросите ручную индексацию в консолях
4. Проверьте наличие ошибок в консолях
