# 🏷️ Исключение товаров Sold Out из подсчета позиций

## Дата: 29 ноября 2025

---

## 🎯 Суть изменения

Товары с бейджем **"Sold Out"** теперь **не учитываются** в количестве позиций рядом с заголовком категории.

---

## 📋 Что изменилось

### До изменения:
```
┌─────────────────────────────────────┐
│ Эспрессо           8 позиций        │
└─────────────────────────────────────┘
```
*Считались все 8 товаров, включая Sold Out*

### После изменения:
```
┌─────────────────────────────────────┐
│ Эспрессо           6 позиций        │
└─────────────────────────────────────┘
```
*Считаются только доступные товары (2 Sold Out исключены)*

---

## 🔧 Технические детали

### Компонент: `GroupedCoffeeTable.tsx`

**Изменена логика подсчета для:**

1. **Зерновой кофе (категории Эспрессо, Фильтр, Молоко и др.)**
   ```typescript
   // Было:
   groupedGrainItems[category].length
   
   // Стало:
   const availableItemsCount = groupedGrainItems[category]
     .filter(item => item.badge !== 'soldout')
     .length;
   ```

2. **Дрип кофе**
   ```typescript
   // Было:
   dripItems.length
   
   // Стало:
   const availableDripItemsCount = dripItems
     .filter(item => item.badge !== 'soldout')
     .length;
   ```

## ✅ Преимущества

### Для пользователей:
- 📊 **Точная информация** - видят реальное количество доступных товаров
- 🎯 **Лучшая навигация** - не вводит в заблуждение по количеству
- 🛍️ **Ясность** - сразу понятно, сколько товаров можно заказать

### Для бизнеса:
- 📈 **Честность** - не завышаем количество доступных товаров
- 🔍 **Прозрачность** - клиенты видят реальную картину
- 💼 **Профессионализм** - правильный учет товарных позиций

---

## 🧪 Тестирование

### Тест-кейсы:

1. **Все товары доступны:**
   - [ ] Счетчик показывает общее количество
   - [ ] Все товары можно добавить в корзину

2. **Один товар Sold Out:**
   - [ ] Счетчик уменьшен на 1
   - [ ] Sold Out товар отображается, но не учитывается

3. **Половина товаров Sold Out:**
   - [ ] Счетчик показывает только доступные
   - [ ] Все товары видны в таблице

4. **Все товары Sold Out:**
   - [ ] Счетчик показывает "0 позиций"
   - [ ] Категория отображается с пустой таблицей

5. **Разные категории:**
   - [ ] Каждая категория считает свои товары отдельно
   - [ ] Sold Out не влияет на другие категории
