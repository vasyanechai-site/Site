# ⚡ Quick Start: Тестирование ссылок на мессенджеры

## 🎯 Что было реализовано

Добавлена функциональность **автоматической генерации кликабельных ссылок** на Telegram и WhatsApp в уведомлениях о регистрации оптовых клиентов.

---

## 🚀 Быстрый тест (2 минуты)

### Вариант 1: Тестовая страница (рекомендуется)

1. Откройте `/messenger-test` в браузере
2. Введите данные:
   - Телефон: `79991234567`
   - Компания: `Тестовая компания`
   - Выберите мессенджер
3. Проверьте предпросмотр сообщения
4. Кликните по сгенерированной ссылке

✅ **Ожидаемый результат:** Откроется выбранный мессенджер с готовым диалогом

### Вариант 2: Реальная регистрация

1. Откройте `/business`
2. Нажмите "Зарегистрироваться"
3. Заполните форму и отправьте
4. Проверьте Telegram-бот

✅ **Ожидаемый результат:** Сообщение с кликабельной ссылкой на мессенджер

---

## 📱 Пример сообщения в Telegram

```
🆕 Новая заявка на регистрацию оптового клиента

📱 Телефон: +79991234567
🏢 Компания: ООО "Тестовая компания"
💬 Мессенджер: Telegram
🔗 Ссылка на Telegram: https://t.me/+79991234567
🕐 Дата: 29.11.2025, 15:30:45
```

**Ссылка синяя и кликабельная** ← Это главное!

## ✅ Что проверить

1. **Форматирование:**
   - [ ] Переносы строк работают (не отображается "\n")
   - [ ] Эмодзи отображаются
   - [ ] Дата в правильном формате

2. **Ссылка:**
   - [ ] Выделена синим (кликабельна)
   - [ ] Открывает мессенджер при клике
   - [ ] Телефон в правильном формате (начинается с +7)

3. **Функциональность:**
   - [ ] Форма регистрации работает
   - [ ] Telegram-бот получает сообщение
   - [ ] Админ-панель показывает заявку

---

## 🔧 Если что-то не работает

### Сообщение не приходит
```bash
# Проверьте переменные окружения
TELEGRAM_BOT_TOKEN=...
TELEGRAM_CHAT_ID=...
```

### Ссылка не кликабельна
- Проверьте, что используется `createRegistrationMessageWithLink` (не старая функция)
- Ссылка должна быть в формате `https://t.me/+79991234567`

### Переносы не работают
- Убедитесь, что в коде используется `\n`, а не `\\n`
- Проверьте функцию `createRegistrationMessageWithLink` в `/supabase/functions/server/index.tsx`
