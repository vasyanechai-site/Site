## ✅ Что настроено в коде

### В `/index.html`:

```html
<!-- Open Graph -->
<meta property="og:image" content="https://coffeenechai.ru/og-image.jpg" />
<meta property="og:image:width" content="1200" />
<meta property="og:image:height" content="630" />

<!-- Twitter Cards -->
<meta name="twitter:card" content="summary_large_image" />
<meta name="twitter:image" content="https://coffeenechai.ru/og-image.jpg" />
```

### В Schema.org JSON-LD:

```json
{
  "@type": "Store",
  "logo": "https://coffeenechai.ru/logo.png",
  "image": "https://coffeenechai.ru/og-image.jpg"
}

{
  "@type": "Organization",
  "logo": "https://coffeenechai.ru/logo.png"
}
```

Все пути правильно указывают на файлы! ✅
