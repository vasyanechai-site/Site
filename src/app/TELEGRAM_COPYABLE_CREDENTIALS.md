# Копируемые учетные данные в Telegram

## Изменения
Обновлено форматирование логина и пароля в Telegram-уведомлениях о заявках на оптовый доступ. Теперь учетные данные можно копировать по клику прямо в Telegram.

## Файлы
- `/supabase/functions/server/index.tsx` - сообщения о заявках
- `/supabase/functions/server/telegram.tsx` - функция отправки (уже использует `parse_mode: 'HTML'`)

## Изменённый код

### Было (Markdown формат)
```typescript
const message = `🆕 *Заявка на получение доступа в оптовый кабинет*\n\n` +
  `👤 *Имя:* \`${name}\`\n` +
  `🏢 *Компания:* \`${company}\`\n` +
  `📞 *Телефон:* \`${phone}\`\n` +
  `💬 *Канал:* ${channelEmoji}\n\n` +
  `🔐 *Данные для входа (созданы автоматически):*\n` +
  `📱 *Логин:* \`${normalizedPhone}\`\n` +
  `🔑 *Пароль:* \`${password}\`\n\n` +
  `💡 _Каждое поле можно скопировать нажатием_`;
```

### Стало (HTML формат)
```typescript
const message = `🆕 <b>Заявка на получение доступа в оптовый кабинет</b>\n\n` +
  `👤 <b>Имя:</b> <code>${name}</code>\n` +
  `🏢 <b>Компания:</b> <code>${company}</code>\n` +
  `📞 <b>Телефон:</b> <code>${phone}</code>\n` +
  `💬 <b>Канал:</b> ${channelEmoji}\n\n` +
  `🔐 <b>Данные для входа (созданы автоматически):</b>\n` +
  `📱 <b>Логин:</b> <pre>${normalizedPhone}</pre>\n` +
  `🔑 <b>Пароль:</b> <pre>${password}</pre>\n\n` +
  `💡 <i>Каждое поле можно скопировать нажатием</i>`;
```

## Объяснение

### HTML теги для Telegram (с parse_mode: 'HTML')

#### `<code>` - инлайн-код
- Используется для обычных полей (имя, компания, телефон)
- Текст становится моноширинным
- Копируется простым выделением

#### `<pre>` - блок кода
- Используется для **логина и пароля** ✅
- Текст становится моноширинным
- **Копируется ПО КЛИКУ** - рядом появляется кнопка "Copy"

#### `<b>` - жирный текст
- Заменяет Markdown `*текст*`

#### `<i>` - курсив
- Заменяет Markdown `_текст_`
