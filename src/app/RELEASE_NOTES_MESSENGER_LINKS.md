# 🚀 Release Notes: Кликабельные ссылки на мессенджеры в уведомлениях

## Дата: 29 ноября 2025

## ✨ Что нового

### 1. Автоматическая генерация ссылок на мессенджеры
Добавлена функциональность автоматического создания кликабельных ссылок на Telegram и WhatsApp в уведомлениях о регистрации оптовых клиентов.

### 2. Исправлено форматирование сообщений
Решена проблема с двойным экранированием символов переноса строк (`\\n` → `\n`), которая приводила к некорректному отображению сообщений в Telegram.

### 3. Улучшенная структура уведомлений
Сообщения теперь включают:
- 📱 Отформатированный номер телефона
- 🏢 Название компании
- 💬 Выбранный мессенджер
- 🔗 **Кликабельная ссылка** для быстрой связи с клиентом
- 🕐 Дата и время заявки

---

## 🛠️ Технические изменения

### Backend (`/supabase/functions/server/index.tsx`)

#### Новые функции:

1. **`formatPhoneForMessenger(phone: string): string`**
   - Очищает номер от лишних символов
   - Конвертирует `8` в `7` для российских номеров
   - Возвращает чистый номер в формате `79991234567`

2. **`generateMessengerLink(phone: string, messenger: string): string`**
   - Генерирует ссылку на Telegram: `https://t.me/+79991234567`
   - Генерирует ссылку на WhatsApp: `https://wa.me/79991234567`

3. **`createRegistrationMessageWithLink(phone: string, companyName: string, messenger: string): string`**
   - Создаёт правильно отформатированное сообщение
   - Использует массив строк и `join('\n')` для корректных переносов
   - Включает кликабельную ссылку на мессенджер

#### Интеграция:
```typescript
// В endpoint /business-registration (строка ~2197)
const message = createRegistrationMessageWithLink(phone, companyName, messenger);
await sendTelegramMessage(message);
```

### Frontend

#### Компоненты без изменений:
- `/components/BusinessRegistration.tsx` - продолжает работать без изменений
- `/components/BusinessPublicPage.tsx` - не требует обновлений

🆕 Новая заявка на регистрацию оптового клиента

📱 Телефон: +79991234567

Ссылка `https://t.me/+79991234567` является **кликабельной** и открывает диалог с клиентом в Telegram.

## 🐛 Известные ограничения

1. **Формат телефона:**
   - Поддерживаются только российские номера (+7)
   - Для международных номеров требуется доработка

2. **Telegram API:**
   - Ссылка `t.me/+number` работает только для номеров, зарегистрированных в Telegram
   - Если номер не зарегистрирован, ссылка не откроется

3. **WhatsApp:**
   - Требуется международный формат без символа `+`
   - Ссылка открывает WhatsApp Web или приложение в зависимости от устройства
