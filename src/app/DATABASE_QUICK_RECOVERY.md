# 🚨 Быстрое восстановление базы данных

Если база данных Supabase показывает проблемы после восстановления:

## 1️⃣ Подождите 5-10 минут

После восстановления проекта Supabase требуется время для инициализации:
- PostgREST может быть "Unhealthy" → подождите
- Auth может быть "Unhealthy" → подождите
- Edge Functions может быть "Unhealthy" → подождите

## 2️⃣ Проверьте состояние через админ-панель

1. Войдите в админ-панель (пароль по умолчанию: `Nechay2024`)
2. Перейдите на вкладку **"Здоровье БД"**
3. Нажмите **"Проверить"**
4. Посмотрите результаты диагностики

## 3️⃣ Что делать если данные потеряны?

### ✅ Автоматически восстановится:
- **Каталог товаров** - создастся при первом запросе
- **Курс доллара** - установится значение по умолчанию (95 ₽/$)

### ❌ Нужно создать заново:
- **Пользователи** - регистрируйте заново
- **Заказы** - история потеряна
- **Промокоды** - создайте через админку

## 4️⃣ Используемая таблица

Приложение хранит все данные в одной таблице: `kv_store_22d84083`

Это key-value хранилище в формате:
```
key (TEXT) | value (JSONB)
```

## 5️⃣ Структура данных в kv_store

| Key Pattern | Описание |
|------------|----------|
| `exchange_rate` | Курс доллара |
| `coffee_items` | Массив товаров |
| `order:XXXXX` | Заказ с номером XXXXX |
| `user:XXXXX` | Пользователь с ID XXXXX |
| `user_settings:XXXXX` | Настройки пользователя |
| `promo:КОД` | Промокод |

## 6️⃣ Быстрая SQL проверка

Проверить наличие данных в Supabase SQL Editor:

```sql
-- Проверить курс доллара
SELECT * FROM kv_store_22d84083 WHERE key = 'exchange_rate';

-- Проверить товары
SELECT * FROM kv_store_22d84083 WHERE key = 'coffee_items';

-- Посмотреть все заказы
SELECT * FROM kv_store_22d84083 WHERE key LIKE 'order:%';

-- Посмотреть всех пользователей
SELECT * FROM kv_store_22d84083 WHERE key LIKE 'user:%';

-- Посчитать количество записей
SELECT 
  COUNT(*) FILTER (WHERE key LIKE 'order:%') as orders_count,
  COUNT(*) FILTER (WHERE key LIKE 'user:%') as users_count,
  COUNT(*) FILTER (WHERE key LIKE 'promo:%') as promo_count
FROM kv_store_22d84083;
```

## 7️⃣ Бэкап данных (на будущее)
