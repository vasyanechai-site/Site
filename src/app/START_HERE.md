# С чего начать (актуально)

Сайт в продакшене ходит в **Node API** (`server/`, обычно URL вида `https://api.…/api`). Фронт собирается **Vite** в папку **`dist/`**.

## Если «не грузится» или пустая страница

1. **Белый экран** — откройте «Просмотр кода страницы»: если там `<script src="/src/main.tsx">`, на хостинг попал **не** продакшен-сборка, а корневой dev-файл. Нужно залить **`dist/`** после `npm run build` (или дождаться успешного **`Deploy Frontend to Reg.ru FTP`**).
2. **Failed to fetch / сеть** — проверьте, что в сборке задан верный **`VITE_API_BASE_URL`** (полный URL с суффиксом `/api`, как в браузере). Откройте в новой вкладке **`…/api/health`** — должен быть JSON с `ok`.
3. **СДЭК / OAuth** — на сайте откройте **`/debug`** → вкладка «СДЭК», проверьте OAuth; на сервере в `.env` должны быть **`CDEK_ACCOUNT`** и **`CDEK_SECRET`**.
