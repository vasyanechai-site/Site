   - Очищает битые символы в POST/PUT/PATCH запросах

## 🛠️ Использование инструментов

### Для разработчиков:

#### 1. Автоматическая проверка при запуске
```typescript
// Запустите приложение в dev режиме
npm run dev

// Откройте консоль браузера
// Вы увидите отчет о проверке кодировки
```

#### 2. Ручная проверка объекта
```typescript
import { checkObjectForBrokenEncoding, generateEncodingReport } from './lib/encodingCheck';

// Проверка объекта
const data = { name: 'Тест', description: 'Описание' };
const issues = checkObjectForBrokenEncoding(data);
console.log('Найдено проблем:', issues.length);

// Получить полный отчет
const { hasIssues, report } = generateEncodingReport(data);
console.log(report);
```

#### 3. Очистка данных перед сохранением
```typescript
import { sanitizeForStorage } from './lib/encodingCheck';

// Автоматическая очистка
const dirtyData = { text: 'Тест с битыми символами' };
const cleanData = sanitizeForStorage(dirtyData);
// cleanData = { text: 'Тест с битыми символами' }
```

#### 4. Проверка в консоли браузера
```javascript
// В development режиме доступна глобальная функция
window.__encodingCheck({ name: 'Тест', text: 'Проверка' });
```
