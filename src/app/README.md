# Nechai Wholesale Coffee

Современный B2B сервис для оптовых заказов кофе.

## Основные возможности

### Для клиентов:
- Каталог с таблицей кофе (управление количеством в кг и пачках 200г)
- Фильтрация по странам происхождения
- Сортировка по цене
- Оформление заказа с автоподсказками организаций (API ФНС через DaData)
- Личный кабинет с историей заказов
- Персональные скидки

### Для администратора:
- Управление прайс-листом
- Просмотр и экспорт заказов
- Управление пользователями и скидками
- Уведомления в Telegram о новых заказах

## Технологии

- **Frontend:** React, TypeScript, Tailwind CSS
- **Backend:** Supabase (Edge Functions, Database, Auth)
- **Шрифт:** Mabry Pro
- **Интеграции:** DaData API, Telegram Bot API

## Запуск проекта

См. инструкции в Figma Make.

## Keep-Alive система

Проект включает автоматическую систему поддержания активности базы данных:

1. **Встроенный механизм** - срабатывает при посещении сайта
2. **GitHub Actions** - запускается каждые 5 дней автоматически

Подробнее: `KEEP_ALIVE_SETUP.md` и `GITHUB_ACTIONS_SETUP.md`
