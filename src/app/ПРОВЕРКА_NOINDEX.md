# ⚡ Быстрая проверка - Удален ли noindex?

## 🔍 Проверка за 2 минуты

### Способ 1: Консоль браузера (РЕКОМЕНДУЕТСЯ)

1. **Откройте сайт:** https://coffeenechai.ru/
2. **Нажмите F12** (откроется DevTools)
3. **Перейдите на вкладку "Console"**
4. **Найдите сообщение:**

#### ✅ Если noindex был и удален:
```
⚠️ НАЙДЕН И УДАЛЕН NOINDEX ТЕГ: noindex, nofollow
✅ NOINDEX УДАЛЕН! Сайт теперь доступен для индексации
```

#### ✅ Если noindex не было:
```
✅ Robots теги установлены правильно - сайт доступен для индексации
```

#### ❌ Если ничего не видно:
- Обновите страницу (F5)
- Очистите консоль и обновите снова
- Проверьте что на вкладке Console, а не Elements

---

### Способ 2: Исходный код (3 секунды)

1. **Откройте сайт:** https://coffeenechai.ru/
2. **Нажмите Ctrl+U** (исходный код)
3. **Нажмите Ctrl+F** и найдите: `noindex`

#### ✅ Правильно:
```html
<meta name="robots" content="index, follow, max-image-preview:large">
<meta name="googlebot" content="index, follow">
<meta name="yandex" content="index, follow">
```

#### ❌ Неправильно (если видите):
```html
<meta name="robots" content="noindex">
<meta name="robots" content="noindex, nofollow">
```

**Примечание:** Если видите noindex в HTML, но в Console написано "NOINDEX УДАЛЕН" - это нормально! React удаляет его при загрузке.

---

### Способ 3: Inspector (F12 → Elements)

1. **Откройте сайт:** https://coffeenechai.ru/
2. **Нажмите F12**
3. **Перейдите на вкладку "Elements"** (или "Элементы")
4. **Найдите `<head>` секцию** (в начале кода)
5. **Найдите meta тег robots**

#### ✅ Должно быть:
```html
<meta name="robots" content="index, follow, max-image-preview:large, max-snippet:-1, max-video-preview:-1">
```

#### ❌ НЕ должно быть:
```html
<meta name="robots" content="noindex">
```

## 🎯 Результат проверки

### ✅ Все хорошо если:
- [x] В Console видно сообщение "сайт доступен для индексации"
- [x] В Elements тег robots содержит "index, follow"
- [x] НЕТ тега с "noindex"

### ⚠️ Проблема если:
- [ ] В Console НЕТ сообщений от SEOHelmet
- [ ] В Elements есть тег с "noindex"

## 🔧 Что делать если проблема

### 1. Очистите кеш браузера
- Нажмите Ctrl+Shift+Delete
- Выберите "Изображения и файлы в кеше"
- Нажмите "Удалить"
- Обновите страницу

### 2. Откройте в режиме инкогнито
- Ctrl+Shift+N (Chrome)
- Ctrl+Shift+P (Firefox)
- Откройте сайт заново
- Проверьте консоль

### 3. Проверьте другие страницы
- Откройте https://coffeenechai.ru/catalog
- Проверьте консоль
- Откройте https://coffeenechai.ru/favorites
- Проверьте консоль
