# Управление точками продаж (Карта)

## Обзор

Добавлен новый раздел "Карта" в админ-панели розницы для управления точками продаж, которые отображаются на странице "Где купить".

## Функциональность

### Админ-панель (/admin → Розница → Карта)

1. **Просмотр всех точек продаж**
   - Отображается список всех точек с названием и адресом
   - Удобная карточная верстка

2. **Добавление новой точки**
   - Кнопка "Добавить адрес"
   - Поля: "Название кофейни" и "Адрес"
   - Автодополнение адресов через DaData API
   - Фильтрация по городу Санкт-Петербург
   - Выпадающий список с подсказками при вводе (от 3 символов)

3. **Редактирование точки**
   - Кнопка редактирования на каждой карточке
   - Те же поля с автодополнением
   - Сохранение/отмена изменений

4. **Удаление точки**
   - Кнопка удаления с подтверждением
   - Точка удаляется из базы данных

5. **Автоматическая инициализация**
   - При первом запуске автоматически загружаются 19 начальных точек продаж
   - Кнопка "Инициализировать данные" доступна если база пуста

## Технические детали

### Backend (Supabase Edge Functions)

**Endpoints:**

1. `POST /make-server-aa167a09/retail-locations/init`
   - Инициализация базы начальными данными
   - Выполняется один раз, проверяет существующие данные

2. `GET /make-server-aa167a09/retail-locations`
   - Получение всех точек продаж

3. `POST /make-server-aa167a09/retail-locations`
   - Добавление новой точки
   - Body: `{ name: string, address: string }`

4. `PUT /make-server-aa167a09/retail-locations/:id`
   - Обновление точки
   - Body: `{ name: string, address: string }`

5. `DELETE /make-server-aa167a09/retail-locations/:id`
   - Удаление точки

**Хранилище:**
- Ключ KV: `nechai_retail_locations`
- Структура: массив объектов `{ id, name, address, createdAt, updatedAt? }`

### Frontend

**Компоненты:**

1. `/components/admin/RetailLocationsManagement.tsx`
   - Основной компонент управления
   - Интеграция с DaData API для автодополнения адресов
   - CRUD операции

2. `/components/LocationsPage.tsx`
   - Обновлен для загрузки данных из API
   - Автоматическая инициализация при первом запуске
   - Fallback на хардкод массив при ошибках

3. `/components/AdminPanel.tsx`
   - Добавлен новый таб "Карта" в разделе "Розница"

## API Integration

**DaData Address Suggestions:**
- API URL: `https://suggestions.dadata.ru/suggestions/api/4_1/rs/suggest/address`
- API Key: использует тот же ключ, что и для поиска компаний
- Фильтр: город Санкт-Петербург
- Количество подсказок: 10

## Стилизация

- Цветовая схема: соответствует дизайн-системе розницы
- Фон выпадающего списка: `#FFF4E5`
- Hover выпадающего списка: `#FFE5CC`
- Акцентный цвет кнопок: розовый `#FF90A1`
- Активная форма: розовая рамка с прозрачным фоном

## Использование

1. Войдите в админ-панель
2. Переключитесь на раздел "Розница"
3. Выберите таб "Карта"
4. Управляйте точками продаж:
   - Добавляйте новые через форму с автодополнением
   - Редактируйте существующие
   - Удаляйте ненужные
5. Изменения сразу отображаются на странице "Где купить" (/locations)
