# Обновлено количество дрип-пакетов в упаковке

## Изменение
Количество дрип-пакетов в упаковке изменено с **5 шт.** на **6 шт.** во всех местах розничного магазина.

## Затронутые файлы

### 1. `/components/RetailProductDetail.tsx` (6 вхождений)
- Обновлены все упоминания `'Упаковка (5 шт.)'` на `'Упаковка (6 шт.)'`
- Функция `getCurrentVariantInCart()` - определение `targetWeight` для drip
- Функция `handleBuy()` - 2 места где устанавливается `weightToAdd`
- Функция `handleIncrement()` - определение `targetWeight`
- Функция `handleDecrement()` - определение `targetWeight`
- UI отображение в селекторе количества для drip-пакетов

### 2. `/components/RetailCartPage.tsx` (1 вхождение)
- Обновлен fallback текст при отображении Badge для drip-пакетов в корзине
- Изменено: `{item.weight || 'Упаковка (5 шт.)'}` → `{item.weight || 'Упаковка (6 шт.)'}`

### 3. `/components/RetailProductsManagement.tsx` (1 вхождение)
- Обновлен label в админ-панели при добавлении/редактировании drip-пакетов
- Изменено: `"Цена за Упаковку 5 шт. (₽)"` → `"Цена за Упаковку 6 шт. (₽)"`

## Результат
Теперь везде в интерфейсе розничного магазина отображается корректная информация:
- На странице товара: "Упаковка (6 шт.)"
- В корзине: "Упаковка (6 шт.)"
- В админ-панели: "Цена за Упаковку 6 шт."

## Совместимость
Изменение обратно совместимо:
- Старые заказы с "Упаковка (5 шт.)" в поле `weight` будут отображаться как есть
- Новые заказы будут создаваться с "Упаковка (6 шт.)"
- База данных не требует миграции, так как это текстовое значение
