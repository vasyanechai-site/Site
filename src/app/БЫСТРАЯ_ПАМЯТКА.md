# ⚡ БЫСТРАЯ ПАМЯТКА: Автоматическая проверка платежей

## 🎉 ГЛАВНОЕ

**Система УЖЕ РАБОТАЕТ автоматически!**  
Никаких действий не требуется! 🚀

---

## ✅ Что происходит автоматически

1. **Через 10 секунд** после загрузки сайта → первая проверка
2. **Каждые 5 минут** → автоматическая проверка всех pending заказов
3. **При входе в ЛК** → проверка заказов пользователя
4. **После оплаты** → проверка конкретного заказа + показ Вушей

---

## 📊 Как проверить что работает

### Вариант 1: Консоль браузера (30 секунд)

1. Откройте сайт
2. F12 → Console
3. Подождите 10 секунд
4. Увидите: `🔄 [Auto] Checking payment statuses...`

### Вариант 2: Тестовый заказ (5 минут)

1. Создайте заказ
2. Оплатите
3. Подождите до 5 минут
4. Статус обновится автоматически!

## ❓ FAQ

### Нужно ли настраивать GitHub Actions?
**НЕТ!** Система работает без GitHub.

### Нужно ли настраивать Supabase Cron?
**НЕТ!** Всё работает в браузере.

### Что если никто не на сайте?
Проверка не идёт. Но как только кто-то зайдёт - все заказы обновятся.

### Нужна проверка 24/7?
Опционально используйте:
- GitHub Actions → `GITHUB_ACTIONS_PAYMENT_CHECKER.md`
- cron-job.org → `SIMPLE_CRON_ALTERNATIVES.md`
