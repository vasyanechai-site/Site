### 2. Google Search Console (10 минут)

**Шаг 1:** Перейдите на https://search.google.com/search-console

**Шаг 2:** Нажмите "Добавить ресурс" → введите `coffeenechai.ru`

**Шаг 3:** Выберите способ подтверждения → "HTML-тег"

**Шаг 4:** Скопируйте код типа:
```html
<meta name="google-site-verification" content="abc123xyz..." />
```

**Шаг 5:** Откройте файл `/index.html` и найдите строку:
```html
<!-- <meta name="google-site-verification" content="YOUR_VERIFICATION_CODE" /> -->
```

**Шаг 6:** Замените на ваш код (уберите `<!--` и `-->`):
```html
<meta name="google-site-verification" content="abc123xyz..." />
```

**Шаг 7:** Сохраните файл и задеплойте изменения

**Шаг 8:** Вернитесь в Google Search Console → нажмите "Подтвердить"

**Шаг 9:** После подтверждения:
- Слева выберите "Файлы Sitemap"
- Добавьте: `https://coffeenechai.ru/sitemap.xml`
- Нажмите "Отправить"

**Шаг 10:** Запросите индексацию:
- Слева выберите "Проверка URL"
- Введите: `https://coffeenechai.ru`
- Нажмите "Запросить индексирование"

## ⚠️ Если через 2 недели сайт не индексируется

### Проверьте доступность файлов:
1. Откройте в браузере: `https://coffeenechai.ru/robots.txt`
   - Должен открыться файл с текстом
   
2. Откройте в браузере: `https://coffeenechai.ru/sitemap.xml`
   - Должен открыться XML файл со списком страниц

### Проверьте отсутствие noindex:
1. Откройте главную страницу сайта
2. Нажмите Ctrl+U (просмотр кода страницы)
3. Нажмите Ctrl+F и найдите: `noindex`
4. Убедитесь что НЕТ строки: `<meta name="robots" content="noindex">`

### Запросите повторную индексацию:
1. Google Search Console → Проверка URL → Запросить индексирование
2. Яндекс.Вебмастер → Инструменты → Переобход страниц
