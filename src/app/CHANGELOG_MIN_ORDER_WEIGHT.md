## 🔧 Техническая реализация

### Функция подсчета веса

Уже существующая функция `getTotalKg()` используется для расчета общего веса:

```typescript
const getTotalKg = (): number => {
  return getCartItems().reduce((sum, item) => {
    // 1 кг + пачки по 200г (5 пачек = 1 кг)
    return sum + item.kg + (item.packs200 * 0.2);
  }, 0);
};
```

**Логика:**
- 1 кг кофе = 1 кг
- 1 пачка 200г = 0.2 кг
- Общий вес = сумма всех килограммов + (количество пачек × 0.2)

---

## 📱 Примеры работы

### Пример 1: Недостаточный вес
**Корзина:**
- 2 кг Kenya AA (2000₽)
- 5 пачек по 200г Rwanda (1000₽)

**Расчет:**
- Общий вес: 2 + (5 × 0.2) = 3 кг ❌
- Общая сумма: 3000₽
- **Результат:** Ошибка "Минимальный заказ 5 кг"

---

### Пример 2: Достаточный вес
**Корзина:**
- 4 кг Ethiopia Sidamo (3200₽)
- 5 пачек по 200г Colombia (1000₽)

**Расчет:**
- Общий вес: 4 + (5 × 0.2) = 5 кг ✅
- Общая сумма: 4200₽
- **Результат:** Заказ можно оформить

---

### Пример 3: Большая сумма, но мало веса
**Корзина:**
- 10 пачек по 200г Geisha (дорогой сорт, 6000₽)

**Расчет:**
- Общий вес: 10 × 0.2 = 2 кг ❌
- Общая сумма: 6000₽
- **Результат:** Ошибка "Минимальный заказ 5 кг" (несмотря на сумму > 5000₽)

### Что осталось без изменений:

1. **Система скидок** - работает по прежнему принципу (от суммы заказа):

### Что изменилось:

1. **Минимальный порог** - теперь по весу, а не по сумме
