# ⚡ Настройка ПРЯМО СЕЙЧАС

## Что нужно сделать (3 шага, 2 минуты)

### ✅ Шаг 1: Создать таблицу в базе данных

1. Откройте [Supabase Dashboard](https://supabase.com/dashboard/project/pkhinqiplfezrzvsqgwo)
2. Нажмите **SQL Editor** в левом меню
3. Создайте новый запрос и вставьте:

```sql
CREATE TABLE telegram_users (
  id BIGINT PRIMARY KEY,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

4. Нажмите **Run** (или Ctrl+Enter)

---

### ✅ Шаг 2: Настроить webhook через админ-панель

1. Откройте **админ-панель** вашего магазина
2. Перейдите на вкладку **"Рассылка"** (Broadcast)
3. В верхнем блоке **"Настройка Webhook"** нажмите **"Установить webhook"**
4. Дождитесь сообщения: ✅ Webhook успешно настроен!

---

### ✅ Шаг 3: Проверить работу

1. Откройте Telegram и найдите вашего бота (тот же бот, который присылает уведомления о заявках)
2. Напишите боту любое сообщение, например: `Привет`
3. Бот должен ответить: **"✅ Вы подписались на рассылку!"**

---

## 🎉 Готово!

Теперь система работает. Вы можете:

### 📤 Отправить рассылку

В админ-панели (вкладка "Рассылка"):
1. Прокрутите вниз до блока **"Отправка рассылки"**
2. Напишите текст сообщения
3. (Опционально) Добавьте изображение
4. Нажмите **"Отправить рассылку"**

Сообщение будет отправлено всем, кто написал боту.

---

## 🔍 Если что-то не работает

### Бот не отвечает на сообщения

**Проверьте:**
1. В админ-панели (раздел "Рассылка") нажмите кнопку обновления (🔄)
2. Убедитесь, что в поле "URL" указан адрес вида:
   ```
   https://pkhinqiplfezrzvsqgwo.supabase.co/functions/v1/telegram-webhook
   ```
3. Если поле пустое или другой URL - нажмите "Установить webhook" еще раз

**Если все равно не работает:**
1. Откройте [Supabase Dashboard](https://supabase.com/dashboard/project/pkhinqiplfezrzvsqgwo)
2. Перейдите в **Functions → telegram-webhook → Logs**
3. Напишите боту сообщение и посмотрите логи
4. Если видите ошибку про таблицу - повторите Шаг 1

---

### Рассылка не отправляется

**Проверьте:**
1. Есть ли у вас подписчики? (Напишите боту хотя бы одно сообщение)
2. В админ-панели должно быть указано количество подписчиков (над кнопкой "Отправить рассылку")

---

### Webhook показывает ошибки

В админ-панели (раздел "Рассылка"):
1. Нажмите **"Удалить webhook"**
2. Подождите 2 секунды
3. Нажмите **"Установить webhook"** заново
4. Нажмите кнопку обновления (🔄) и проверьте, что ошибок нет

---

## 💡 Альтернативный способ (через командную строку)

Если админ-панель не работает, можно настроить webhook вручную:

### Через Node.js:
```bash
node docs/telegram/setup-webhook.js <ВАШ_BOT_TOKEN>
```

### Через Bash:
```bash
chmod +x docs/telegram/setup-webhook.sh
./docs/telegram/setup-webhook.sh <ВАШ_BOT_TOKEN>
```

Токен бота можно найти в Supabase Dashboard → Settings → Edge Functions → Environment Variables → `TELEGRAM_BOT_TOKEN`
