```
Должно быть ТОЧНО: TOCHKA_JWT_TOKEN
Не: tochka_jwt_token
Не: TochkaJwtToken
Не: TOCHKA JWT TOKEN
```
☐ Проверил, имя правильное

#### Проверка 2: Значение токена
```
Должно начинаться с: eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9
Длина: около 572 символов
Нет пробелов в начале/конце
```
☐ Проверил, токен скопирован полностью

#### Проверка 3: Время ожидания
```
После сохранения Secrets подождал: ⏱ 30-60 секунд
Повторил диагностику
```
☐ Подождал и повторил

### ❌ Проблема: "JWT токен слишком короткий"

```
Показывает: Длина: 150 символов (ожидается ~572)
```

**Решение:**
1. ☐ Откройте исходный токен
2. ☐ Выделите ВСЁ от начала до конца
3. ☐ Скопируйте (`Ctrl+C` или `Cmd+C`)
4. ☐ Вставьте в Supabase Secrets
5. ☐ Убедитесь, что нет обрезания
6. ☐ Сохраните
7. ☐ Подождите 60 секунд
8. ☐ Повторите диагностику
